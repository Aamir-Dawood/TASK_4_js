How It Works

Click on any heart icon to toggle between outline and filled states:

  • ♡ - Outline heart (not favorited)
  • ❤ - Filled heart (favorited)
// JavaScript logic:
const buttons = document.querySelectorAll('.favorite-icon');
buttons.forEach(btn => {
  btn.addEventListener('click', () => {
    btn.classList.toggle('filled');
    btn.innerHTML = btn.classList.contains('filled') ? '❤' : '♡';
  });
});
